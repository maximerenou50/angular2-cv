About this website

On this page, I will explain how I built my website and what technologies I chose.

Technologies

Here is the technologies I used to build my website:

  • Angular 2: Angular is the famous JavaScript framework of Google. I use the lastest version to manage my application. You can choose to build it with JavaScript or TypeScript. TypeScript is a object-oriented programming language developed by Microsoft and a superset of JavaScript, that what I chose for my website.
  • Meteor: Meteor (more specifically Angular Meteor) is a framework which let you manage both client and server side using Angular. It also offer an access to MongoDB and a lot of functionalities.
  • Bootstrap: Bootstrap is the most famous CSS framework built by twitter. It's easy to make a nice web app with it but you have to override the default design to look different from other websites.
  • MongoDB: MongoDB is a NoSQL database, document-oriented. I was looking for a NoSQL database, because I was searching simplicity and flexibilty, and I was looking for a document-oriented database because my app manage complex type and this kind of NoSQL was the more appropriate for my app.

Hosting

To host my application, I used:

  • Galaxy: Galaxy is the platform as a service for Meteor applications. This service is host on AWS and use Docker to deploy apps. I used this service because of its simplicity, it's very easy to deploy an app with it and I was looking for a PaaS to host my application.
  • mLab: mLab is a database as a service for MongoDB. It's also hosted on AWS and has a free offer for small application such as mine.
  • 1&1: 1&1 is an hosting, a domain registration and a website services. I only used the domain registration to buy maximerenou50.com and I used their SMTP service to send me email through the contact form.

Responsive design

My web application is fully responsive and works on computers, tablets and smartphones thanks to Bootstrap. Because I worked with mobile technologies before, I make sure my app will work on all devices, not just computers. I'm also concern about cross-mobile issues and that another reason why I love Meteor.
